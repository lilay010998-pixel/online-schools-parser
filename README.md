# 🎓 Online Schools Parser

Парсер для сбора данных об онлайн-школах России с ОКВЭД кодом 85.42 и похожими.

## 📋 Возможности

✅ Сбор информации об онлайн-школах:
- Название компании
- Описание деятельности
- Официальный сайт
- Годовой оборот (от 50 млн руб)
- ФИО владельца и его контакты
- Ссылки на соцсети
- Ключевые сотрудники
- Вакансии на HH.ru

## 🗂️ Структура проекта

```
online-schools-parser/
├── src/
│   ├── scrapers/
│   │   ├── website_parser.py       # Парсинг сайтов
│   │   └── hh_parser.py            # Парсинг вакансий HH.ru
│   ├── processors/
│   │   ├── data_processor.py       # Обработка данных
│   │   └── excel_exporter.py       # Экспорт в Excel
│   └── main.py                     # Главный скрипт
├── config.py                       # Конфигурация
├── requirements.txt                # Зависимости
└── output/
    └── schools_database.xlsx       # Результат (автоматически создаётся)
```

## 🛠️ Установка

### Требования
- Python 3.8+
- pip

### Шаги установки

1. **Клонируй репозиторий:**
```bash
git clone https://github.com/lilay010998-pixel/online-schools-parser.git
cd online-schools-parser
```

2. **Установи зависимости:**
```bash
pip install -r requirements.txt
```

## 🚀 Использование

### Базовый запуск

```bash
python src/main.py
```

### Результат
Скрипт создаст Excel файл `output/schools_database.xlsx` с данными всех найденных школ.

## 📊 Данные в Excel

Файл содержит следующие колонки:

| Колонка | Описание |
|---------|---------|
| Название компании | Полное наименование |
| ОКВЭД код | Классификационный код |
| Тип деятельности | Категория школы |
| Описание (что учит) | Направления обучения |
| Сайт | URL официального сайта |
| Годовой оборот | Объём продаж в год |
| ФИО владельца | Руководитель компании |
| Контакты владельца | Телефоны, email |
| Соцсети владельца | Ссылки на соцсети |
| Ключевые сотрудники | ФИО и должности |
| Вакансии (HH.ru) | Ссылка на вакансии |
| Источник информации | Откуда собраны данные |
| Дата добавления | Дата парсинга |

## 🔍 Источники данных

Парсер собирает информацию из:

1. **Яндекс.Карты** - профили компаний и рейтинги
2. **Сайты школ** - описание программ и контакты
3. **HH.ru API** - актуальные вакансии
4. **Google Поиск** - информация о владельцах и контактах
5. **LinkedIn** - данные о сотрудниках (где доступно)

## ⚙️ Конфигурация

В файле `config.py` можно настроить:

```python

# Максимум компаний на регион

# Регионы для парсинга

    "Москва",
    "Санкт-Петербург",
    "Екатеринбург",
    # ... и другие
]
```

## 🐛 Решение проблем

### Проблема: "ModuleNotFoundError"
**Решение:** Убедись, что ты в корневой папке проекта и зависимости установлены:
```bash
pip install -r requirements.txt
```

### Проблема: Excel файл не создался
**Решение:** Проверь, что папка `output` существует или создана автоматически.

### Проблема: Ошибки при парсинге
**Решение:** Некоторые сайты могут блокировать парсеры. Это нормально, парсер пропускает недоступные источники.

- С контактами владельца
- С вакансиями
